Uma empresa ficticia chamada "TechFlow" sofreu um ciberataque. O único rasto deixado pelo hacker foi uma caneta USB.
Acredita-se que essa caneta USB possa conter evidências críticas que podem ajudar a identificar o hacker.
A caneta encontra-se organizada por pastas. Cada pasta contêm uma pista na forma de uma flag que deve ser recuperada.
